# 🎮 CubyRace

**A rage-inducing platformer that tests your patience and precision.**  
You control a small cube that must reach the end **without touching any obstacles**

---

## 🕹️ Controls
- **A** → Move Left  
- **D** → Move Right  

Simple controls. Extreme difficulty. 💢
